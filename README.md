# Sistema de Chamados (Help Desk)

Projeto desenvolvido em Java simulando um sistema de suporte técnico (Help Desk), com criação e gerenciamento de chamados.

## Funcionalidades
- Criar chamados
- Listar chamados
- Atualizar status (Aberto, Em andamento, Resolvido)

## Tecnologias utilizadas
- Java
- Lógica de programação
- Estrutura de dados

## Objetivo
Simular um ambiente real de suporte técnico, aplicando conceitos utilizados em sistemas ERP e atendimento ao usuário.
